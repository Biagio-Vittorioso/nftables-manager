# NFTables Manager

Frontend grafico open source per gestire nftables su Linux.
Scritta in Rust + GTK4 + Libadwaita.

Caratteristiche:
- Sidebar Tables/Chains
- Tabella regole: Source, Destination, Protocol, Port, Action
- Contatori traffico: Packets/Bytes
- Add/Edit/Delete rules via dialog GTK
- Drag & Drop regole
- Reload automatico
- Icona scudo nel launcher
